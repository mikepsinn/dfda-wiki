The proposed solution is the decentralized FDA: a shared, open, continuously learning system for ranking treatments, running pragmatic trials, and turning real-world evidence into better decisions. The maintained explanation of that solution now lives on the War on Disease manual.

This wiki keeps a short bridge page because the repo still contains the engineering and policy work needed to make the idea concrete. If you want to revise the high-level vision, update the manual. If you want the build plan, interfaces, and implementation details, use the local docs below.
